# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of miniAI seriously. 
If you discover a security vulnerability, please report it by emailing [nelsonramos@ua.pt].  
Please, do **not** create public issues for security concerns.

## Security Best Practices

- Keep dependencies up to date.
- Review code for potential vulnerabilities.
- Limit access to sensitive data and credentials.

## Responsible Disclosure

We appreciate responsible disclosure and will respond promptly to reports.
